'''B4X'''[https://www.b4x.com/] is a programming language and a set of cross-platform RAD development tools that allow complete beginners, citizen developers, and professionals to build real-world Android, iOS and desktop solutions. B4X is a set of simple and powerful cross platform RAD tool: B4A (free)[https://www.b4x.com/b4a.html] for Android development, B4J (free)[https://www.b4x.com/b4j.html] for Desktop, Server and Web development, B4i [https://www.b4x.com/b4i.html] for iOS development and B4R (free)[https://www.b4x.com/b4r.html] for Arduino, ESP8266 and ESP32 development. It's a very easy to learn language[https://www.b4x.com/learn.html]. You have some question or problem? Don't worry, you have a very active community[https://www.b4x.com/android/forum/].
