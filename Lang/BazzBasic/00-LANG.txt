'''BazzBasic''' is a free/[[open source]] ([[MIT]]), modern [[BASIC]] interpreter for [[Microsoft]] [[Windows]], built on [[.NET]] 10 with [[SDL2]] graphics and [[SDL2_mixer]] sound support. It is not a clone of any previous BASIC — it aims to be easy, fun, and expressive.

BazzBasic uses a typeless variable model where all variables and constants are declared with sigil suffixes: <code>$</code> for mutable variables and <code>#</code> for immutable constants. Variables can hold numbers or strings interchangeably. The language supports user-defined functions with isolated scope, dynamic arrays with numeric, string, or multi-dimensional indexing, associative arrays, and first-class label strings enabling dynamic GOTO and GOSUB calls.

Built-in support includes SHA-256 hashing, HTTP/HTTPS via <code>HTTPGET</code>/<code>HTTPPOST</code>, JSON encoding and decoding via <code>ASJSON</code>/<code>ASARRAY</code>, file I/O, system time formatting, and a full SDL2 graphics and sprite system. Fast trigonometry lookup tables are available for graphics-intensive applications.

The interpreter is distributed as a single Windows executable with a built-in IDE (syntax highlighting, find/replace, tabbed editing). Programs can be compiled to standalone <code>.exe</code> files or tokenized <code>.bb</code> libraries.
