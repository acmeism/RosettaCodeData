'''Hobbes''' is a statically-typed, eagerly-evaluated functional programming language that compiles to native machine code via LLVM JIT. Originally developed at [[wp:Morgan Stanley|Morgan Stanley]], it is designed for high-performance structured data processing in low-latency environments.

Key features include:
* Hindley-Milner type inference (type annotations are rarely needed)
* Algebraic data types (records, variants, tuples)
* Pattern matching with guards and regex support
* List comprehensions
* Haskell-style type classes with functional dependencies
* LLVM JIT compilation to native code
* Inline LALR(1) parser generator
* C++ interop for embedding in applications

Hobbes provides an interactive REPL called <code>hi</code> and supports script files with the <code>.hob</code> extension.
