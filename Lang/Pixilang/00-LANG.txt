'''Pixilang''' is a "pixel-oriented" programming language designed to facilitate small sound and graphics programs and experiments, currently available for Windows 2000 and up, macOS 10.9 and up, Linux (x86, x86_64, ARM (Raspberry Pi, PocketCHIP, etc.), ARM64 (PINE64, etc.)), iOS 9.0 and up (source only), Android 4.1 and up and Windows CE (Pocket PC and Windows Mobile, ARM only). Programs are stored in plain text, using either the .pixi or .txt file extension.
