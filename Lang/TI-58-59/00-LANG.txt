The TI-59, along with its scaled-down TI-58 and TI-58C versions, was a programmable calculator manufactured by Texas Instruments between 1977 and 1983. It had up to 960 program steps and 100 memory registers, with a flexible partitioning system that allowed 10 memory registers to be "traded" for 80 program steps. It managed single-precision floating-point numbers.

The programming language is like a primitive macro assembler.  Unlike the TI-57, complex instructions used multiple steps: storing something in register #1 had the same syntax (STO 1) for both machines, but it counted as 2 steps by the TI-59. It had elaborate addressing possibilities: absolute (step index), relative (label) and indirect (step index contained in a specified memory register). 

For flow control, it had 2 tests comparing the value of the display (the x register) with a dedicated register (the t register, only accessible by an instruction exchanging its value with x), 10 flags, 2 instructions for incrementing or decrementing registers with jumps to zero, unconditional branch (GTO) and subroutines with a dedicated stack.

Target users were high school students and engineers, so basic mathematical, trigonometric and statistical functions were available.
