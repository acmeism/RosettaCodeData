;Definition 
A word is an &nbsp; ''ABC word'' &nbsp; if the letters &nbsp;"'''a'''", &nbsp;"'''b'''"&nbsp; and &nbsp;"'''c'''"&nbsp; appear in the word in alphabetical order.

If any or all of these letters occur more than once in a word, then only the first occurrence of each letter should be used to determine whether a word is an &nbsp;''ABC word''&nbsp; or not.
