It is conjectured that each decimal digit will occur with the same frequency in the mathematical constant e, see:  [[wp:E_(mathematical_constant)|the Wikipedia page on e]].<br/>
This task is to count and display (here on this page) how many times each digit occurs, up to some "large" number of fractional digits.

;Notes

A [[B]] program to calculate the first 4000 fractional digits is on the Wikipedia here: [[wp:B_(programming_language)|B (programming language)]]<br/>
The algorithm shown is from Ken Thompson's ''Users' Reference to B'' manual and does not require big-integers.
