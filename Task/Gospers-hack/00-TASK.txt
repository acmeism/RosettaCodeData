'''Gosper's Hack''' is an efficient method of computing the next higher integer with the same number of set bits (Hamming weight). It was invented by [https://en.wikipedia.org/wiki/Bill_Gosper Bill Gosper]. The algorithm is widely used in combinatorics for enumerating combinations.

;Task

Write a function or procedure to compute the next higher integer with the same number of set bits as a given positive integer, using Gosper’s Hack.

Show the result of applying it ten times to each of the numbers 1, 3, 7, and 15.
