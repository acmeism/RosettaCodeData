;Task:
Create a program that computes an approximation of the principal &nbsp; N<sup>th</sup> &nbsp; root of &nbsp; X &nbsp; as the largest integer less than or equal to &nbsp; R &nbsp; for which &nbsp; R<sup>N</sup>=X. 

──where:
        N  is a positive integer. 
        X  is a non-negative integer. 
        R  (the root) &nbsp; is a non-negative real number. 

No arbitrary limits should be placed on the magnitudes of the numbers involved.


Example: &nbsp; With &nbsp; N=3 &nbsp; and &nbsp; X=8 &nbsp; you would calculate the number &nbsp; '''2''' &nbsp; because &nbsp; <big><math>2^3=8</math></big>

Example: &nbsp; With &nbsp; N=3 &nbsp; and &nbsp; X=9&nbsp;  you would again calculate the number &nbsp; '''2''' &nbsp; because '''2''' is the largest integer less than or equal to the root &nbsp; R.

Example: &nbsp; With &nbsp; N=2 &nbsp; and &nbsp; X=2&times;100<sup>2,000</sup> &nbsp; you would calculate a large integer consisting of the first &nbsp; 2,001 &nbsp; digits (in order) of the square root of two.
