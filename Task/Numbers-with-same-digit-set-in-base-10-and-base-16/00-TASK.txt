;Task
Find decimal numbers &nbsp; '''n''' &nbsp; that when converted to '''hexadecimal''' produce a number that uses the same set of digits (regardless of order and ignoring duplicates) as the original number, &nbsp; where &nbsp; '''n &nbsp;&lt;&nbsp; 100000'''


;Example:
The decimal number &nbsp; '''2339''' &nbsp; is &nbsp; '''923''' &nbsp; when written in hexadecimal. 

The set of digits used, &nbsp; ignoring order and duplicates, &nbsp; is &nbsp; '''{2, 3, 9}''' &nbsp; in both cases and hence this number satisfies the task requirements.
