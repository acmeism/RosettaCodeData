;Task:
Write a program that can parse a grammar in &nbsp; Extended Backus–Naur Form &nbsp; ('''EBNF'''), &nbsp; and then parse something else according to the grammar. 

The program is only required to decide whether or not the something else belongs to the language described by the grammar, &nbsp;but for extra credit, &nbsp; it can output a syntax tree. 

See [[Parse EBNF/Tests|the tests]].
