Rational number arithmetic is exact for four arithmetic operations +,-,*,/. The objective of this task is to create a rational number calculator. The calculator must support '''+''','''-''','''*''','''/''', unary '''+''','''-''','''abs''' (absolute value) and ordering parenthesis: 1 + (2 - 7) / 5. Use '''@''' to denote the previous result.

You are free to use any tools for parsing (e.g. YACC, LEX etc) or numeric libraries your language has access to.

Bonus task. Support recurring notation e.g. 1/3 = 0.(3). Note that 0.(9) = 0.(99) = 1. Also note the cases when the denominator is not a co-prime of 10.
