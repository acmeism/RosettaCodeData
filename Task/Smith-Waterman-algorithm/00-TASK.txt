[https://en.wikipedia.org/wiki/Smith%E2%80%93Waterman_algorithm The Smith–Waterman algorithm] is a dynamic programming algorithm used for local sequence alignment. It identifies similar regions between two sequences by comparing segments and aligning them to optimize a similarity score. This algorithm is widely used in bioinformatics for comparing DNA, RNA, or protein sequences.

;Problem Description

Given two sequences, <math>A = a_1, a_2, ..., a_m</math> and <math>B = b_1, b_2, ..., b_n</math>, the goal is to find the optimal local alignment between these sequences. A local alignment aligns a substring of <math>A</math> with a substring of <math>B</math> to maximize a similarity score.

;Scoring System

The algorithm uses the following scoring system:
* <math>+s</math> for a match between two characters.
* <math>-d</math> for a mismatch (substitution).
* <math>-g</math> for a gap (insertion or deletion).

;Dynamic Programming Matrix

The algorithm constructs a dynamic programming matrix <math>H</math> of dimensions <math>(m+1) \times (n+1)</math>, where <math>H(i, j)</math> represents the optimal alignment score between substrings <math>A[1..i]</math> and <math>B[1..j]</math>. The matrix is initialized as:

<math>
H(i, 0) = 0 \quad \text{for all } i \in [0, m]
</math>
<math>
H(0, j) = 0 \quad \text{for all } j \in [0, n]
</math>

The recurrence relation used to compute the score is:

<math>
H(i, j) = \max \begin{cases} 
    0, \\
    H(i-1, j-1) + \text{score}(a_i, b_j), \\
    H(i-1, j) - g, \\
    H(i, j-1) - g
\end{cases}
</math>

Here, <math>\text{score}(a_i, b_j)</math> is <math>+s</math> if <math>a_i = b_j</math> (match) and <math>-d</math> otherwise (mismatch).

;Traceback

To determine the optimal local alignment, traceback is performed starting from the cell in <math>H</math> with the maximum value. The traceback continues until a cell with a value of <math>0</math> is reached, indicating the end of the local alignment.

;Output

The output of the algorithm includes:
* The highest alignment score.
* The aligned substrings of <math>A</math> and <math>B</math>.
* The alignment path in the dynamic programming matrix.

;Example

Consider the two sequences <code>A = "ACACACTA"</code> and <code>B = "AGCACACA"</code>. Using a match score of <math>+2</math>, mismatch penalty of <math>-1</math>, and gap penalty of <math>-2</math>, the Smith–Waterman algorithm identifies the optimal local alignment with a score of <math>14</math>, aligning the substrings <code>"ACACACT"</code> from <code>A</code> and <code>"GCACACA"</code> from <math>B</math>.
