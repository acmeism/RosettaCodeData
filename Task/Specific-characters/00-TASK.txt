Given a list of strings.

A character in each string is specific if:
# It appears exactly twice in this string.
# It does not appear in any other string in the list.
Answer the question: How many ''distinct'' specific characters are in the list?

(That is, if a character satisfies the condition, it is only counted once.)

For example, ["ahwiueshaiu","ajxxfioaaf","ajrdsfroiwr"] -> [2, 1, 0].

You can assume that all inputs are in ASCII.

<hr/>
'''Extra credit:''' count the ''distinct'' characters which do not satisfy the condition.

For example, the above input -> [5, 5, 9].

(Hint: this part is the number of unique characters in each string that do not pass muster, so the first 5 does not count a and i twice, or, of course, h and u at all, and the last 9 does not count r thrice. Should that still not make sense, you could just translate the Phix answer.)
