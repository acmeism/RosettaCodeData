Vile and Dopey numbers are complementary sets containing all positive integers. (Much like Evil and Odious numbers.) 

Vile numbers are the set of integers where if some integer '''n''' is in the set, then '''2n''' is not. '''2n''' would be classified as a Dopey number.

Conversely, For each Dopey number '''m''', '''2m''' is a Vile number.



Starting with '''1''', there is no integer 1/2, so '''1''' is a Vile number.

'''2''' is 2 times 1 so it can't be a Vile number; it must be Dopey.

'''3''' has no integer half its value, so Vile.

'''4''' has an integer half its value, '''2''',  but '''2''' is Dopey, so '''4''' must be Vile.

''and so on...''


Since if '''n''' is a Vile number, and '''2n''' is a Dopey number, odd numbers must all be Vile. Even numbers ''may'' be Dopey, but ''may'' be Vile.

There are approximately twice as many Vile numbers as Dopey. 

An easy way to tell if a number is Vile or Dopey is to look at the binary representation of the number, and see if it ends with an even or odd number of zeros.

Vile numbers end with an even number of zeros, Dopey numbers with odd.
