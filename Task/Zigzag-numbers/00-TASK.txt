: Zigzag Numbers

A permutation of a set {1, ..., n} is a rearrangement of the elements of the set.
A permutation c(1), ..., c(n) is said to be ''zigzag'' or alternating if its entries 
alternately rise and descend. Because the values in the permutation rise and fall 
back and forth, the first entry c(1) is always less than the second c(2), and 
similarly each entry other than the first and the last should be either larger 
or smaller than both of its neighbors. Here, we will use the term ''zigzag''
to refer to the permutations that alternate ''up-down'' at their start in this way, 
with the first entry less than the second entry. There are also ''down-up'' permutations 
that alternate in the opposite way, with the first entry greater than the second entry.

There is a one-to-one correspondence between the down-up and up-down permutations. If you 
replace each entry c(i) with n + 1 - c(i), this creates a corresponding ''down-up'' permutation.

André's theorem states that the number of up-down permutations of length n is equal to the 
number of down-up permutations of length n, as you would expect since using that replacement  
creates a one-to-one bijection between the two types of permutations.

Determination of the number A(n) of zigzag permutations of the set {1, ..., n} is called 
André's problem. The numbers An are variously known as Euler numbers, or ''zigzag numbers''.

Sets of order 0 or 1 are defined to have exactly one zigzag permutation, so A(0) = A(1) = 1.


;Task: 
Write a function that generates all zigzag permutations of the set {1, ..., n}
and a function that returns the number of zigzag permutations of the set {1, ..., m}. 

Show your results for the above using ''n'' from 1 to 5 and ''m'' from 1 to 30.
